### 전역 스타일

### 컴포넌트 CSS

- 컴포넌트 레벨의 CSS추가 가능
  - 다른 컴포넌트 클래스와 충돌나지않게 고유 클래스명 제공

### SCSS와 SASS

- SCSS와 SASS 두가지 경우 모두 css를 사용할때와 마찬가지로 동일한 방식으로 사용가능

**SCSS**

- `npm install--save-dev sass` 와 같은 명령어로 설치가능
- 제공하는 variable 컴포넌트사용시 export문법 사용

### CSS-in-JS

- styled-component에서만 다룸
  - 이외의 방법은 공식문서에 방법이 있음
