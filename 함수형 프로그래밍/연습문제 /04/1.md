### 추철하는단계

1. 계산 분리

```js
function update_tax_dom() {
  set_tax_dom(calc_tax());
}
function calc_tax() {
  return shopping_cart_total * 0.1;
}
```

2. 암묵적 입력 인자로 변경

```js
function update_tax_dom() {
  set_tax_dom(calc_tax(shopping_cart_total));
}
function calc_tax(tax) {
  return tax * 0.1;
}
```
